# Virent — Быстрый запуск на своём ПК

## 1. Скачать и установить

Скачай `VirentSetup.exe` из [GitHub Releases](https://github.com/FreedoomForm/Virent/releases).

Установи. При первом запуске:
- Сервер стартует на `http://localhost:8443`
- Ngrok поднимает туннель: `https://caliber-lividly-coastline.ngrok-free.dev`

## 2. Подключить телефон

1. Установи APK на Android-телефон
2. Открой приложение → ввод номера → выбери флаг 🇺🇿
3. OTP придёт с админского телефона (SMS-шлюз)
4. Карта Ташкента работает сразу (тайлы в APK)

## 3. Админ-панель

Открой в браузере или в Windows-приложении:
- Логин: `+998900000001` (без OTP)
- Все 54 страницы админки работают локально

## 4. Маршруты (опционально)

```bash
bash osrm-setup/setup-local-osrm.sh
```
OSRM Docker на `localhost:5000` — маршруты по реальным дорогам.

## 5. Бэкап

В админке: Настройки → Бэкап → Sync to SQLite.
Или авто-бэкап каждые 30 секунд.

## Системные требования

- Windows 10/11, 8 GB RAM
- Docker (опционально, для OSRM)
- Android 8+ телефон с SIM-картой
