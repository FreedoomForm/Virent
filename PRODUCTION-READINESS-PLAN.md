```

└── RECEIVE_SMS permission

```


```
└── Экран в админке: выбор SIM, ручной ввод номера

```

### 2.4 Тест end-to-end
```
└── Пользователь регистрируется → получает OTP
└── OTP отправлен с SIM-карты админского телефона
└── Поток: регистрация → OTP → вход → карта → поездка
```
