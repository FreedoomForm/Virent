# Competitor Analysis — ViRent (cloud.togotech.virent)

## 24 screenshots analyzed — key features to adopt:

### 1. Auth Flow
- **Telegram login** as alternative to phone OTP (one-tap login)
- **Language selector on auth screen** (flag + dropdown, visible from start)
- **SMS countdown timer** with "Send again after 0:32" (prevents spam)
- Dark theme auth with scooter illustration

### 2. Map Screen
- **"Show scooters nearby" floating button** (green, prominent)
- **QR scanner button** at bottom center (always accessible)
- **Location target button** (recenter map on user)
- **Hamburger menu + logo + notification bell + settings** in app bar
- **Scooter clustering** when zoomed out (shows density)
- **Map style options**: beige, dark blue, white (user preference)

### 3. Map Settings Page (NEW — we don't have this)
- Toggle: "Show rental zone", "Show finish zone", "Show geopoints"
- Slider: "Minimum charge level" (filter scooters by battery %)
- Map style picker (3 options)
- Language selector
- FAQ link

### 4. Scooter Booking Modal (NEW — better than ours)
- Scooter image, ID, battery %, range (km), insured status
- **Tariff comparison**: per-minute vs 30-minute (with price breakdown)
- Collapsible sections: Tariff, Payment, Promocode
- "Book" (outline) + "Let's go" (filled) dual buttons
- Pre-selected payment method

### 5. Payment Management
- Saved cards with bank logo + masked number
- Green border on active/selected card
- "Add payment method +" with green text
- Card entry modal: number, validity, CVC, cardholder, phone

### 6. Ride History
- Card per ride: date, time, scooter ID, duration, distance, cost
- **"Track ride"** button (see route on map)
- **"Receipts"** button (download/view receipt)

### 7. Notifications
- "X new notifications" counter in header
- Green bell icons, notification type colors
- Truncated text (expandable)

### 8. Help/Support Page (better than ours)
- Social media icons (Instagram, Facebook, Telegram, Phone)
- Email link
- "User's Manual" link
- "FAQ" link
- **"Write to support"** prominent CTA button

### 9. FAQ Page (NEW — we don't have this)
- Expandable/collapsible question cards
- Categories: Scooter rental, Damage, Tariffs, etc.
- Detailed answers with legal compliance (age, weight, helmet, road rules)
- "+" to expand, "-" to collapse

### 10. Design Patterns to Adopt
- **Dark theme** with green accents (primary CTA color)
- **Collapsible sections** in modals (tariff/payment/promo)
- **Tariff comparison** (per-minute vs package deals)
- **Scooter range display** (km on battery %)
- **Insurance status** on scooter info
- **Map style customization**
- **Minimum battery filter slider**
- **Ride tracking** (replay route on map)
- **Receipts** (downloadable ride receipts)
- **Social media support channels**
