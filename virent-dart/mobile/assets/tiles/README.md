# Pre-bundled map tiles for Tashkent

Tiles are included in the APK so the map works immediately without internet.

Format: assets/tiles/tashkent/{z}/{x}/{y}.png
Zooms: 12–16
Area: Tashkent city center (41.23–41.38 N, 69.15–69.35 E)

After first launch, flutter_map additionally caches tiles it fetches,
so the map only gets better over time.
